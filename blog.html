Important FAQ

What is the document object model (DOM)?

ANS: The Document Object Model (DOM) is a programming interface that represents the structure of an HTML or XML document as a tree of objects, allowing programs to dynamically access, modify, and manipulate the document's content, structure, and style.

How do you select an element from DOM?

ANS: To select an element from the DOM, I can use JavaScript methods like document.getElementById() to select by ID, document.querySelector() to select the first element that matches a CSS selector, or other methods like document.getElementsByClassName() and document.getElementsByTagName() for selecting by class or tag name.

What is event delegation in the context of the DOM, and why it is useful?

ANS: Event delegation is a technique in the DOM where a single event listener is attached to a parent element to manage events on its child elements. Instead of adding individual listeners to each child, the event is captured and handled by the parent, taking advantage of event bubbling. This is useful for improving performance and managing dynamic elements added to the DOM.

How do you manipulate an element's attributes and style using the DOM?

ANS: I can manipulate an element's attributes and style using the DOM by accessing the element via methods like document.getElementById() or document.querySelector(). To change attributes, use methods like setAttribute() or directly modify properties like element.src or element.href. To modify styles, use the style property (e.g., element.style.color = "red").
